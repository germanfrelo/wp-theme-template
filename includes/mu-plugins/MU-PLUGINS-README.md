# Must-use plugin to customize the site login page

Files included:

- `login-page.php`
- `login-page.css`

Steps:

1. Copy both files to the `wp-content/mu-plugins/` folder on the site.
2. Add the site logo to the same folder and name it `login-logo.svg` (sanitize it first using `svgo` or similar tool).
3. Check the page `/wp-login.php` in the browser to verify.
