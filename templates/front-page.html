Paragraph

HTML paragraphs can be any structural grouping of related content, such as images or form fields.

Breaking up content into paragraphs helps make a page more accessible. Screen-readers and other assistive technology provide shortcuts to let their users skip to the next or previous paragraph, letting them skim content like how white space lets visual users skip around.

Media & Text

Media on left — above when stacked on mobile

The Media Object is a pattern we see all over the web. It refers to a two-column box with an image on one side and descriptive text on the other, e.g. a social media post.

Media Object pattern needs some or all of the following characteristics:

  • Stacked on Mobile, two columns on Desktop.
  • The image can be on the left or right.
  • The image might be small or large.
  • Media Objects can be nested.
  • The Media Object should clear the contents no matter which side is tallest.
