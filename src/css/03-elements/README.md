
# Elements

 Low-specificity, global styles for HTML elements that apply to the whole project.

Global custom properties are defined in files in [`02-tokens/`](./../02-tokens/),
allowing us to customize globals with design tokens, rather than
writing the same styles for *every* project.

## Organization

Directories:

01. `01-root/`: Root/body element and global pseudo-elements/classes.
02. `02-inline-level/`: Inline-level elements.
03. `03-block-level/`: Block-level elements.
